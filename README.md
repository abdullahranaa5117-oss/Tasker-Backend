# Tasker — Backend

Node.js REST API with JWT auth and MongoDB Atlas.

# Tech Stack 
**Runtime Environment:** Node.js.

**Framework:** Express.js.

**Database:** MongoDB Atlas.

**Authentication:** JSON WEB TOKEN & Bcrypt

## Setup

git clone https://github.com/abdullahranaa5117-oss/Tasker-Backend

cd tasker-backend

npm install

mkdir uploads

# env

DB_URI=

JWT_SECRET_KEY=

CLIENT_URL=

PORT=5000

# Running the Server

npm run debug
